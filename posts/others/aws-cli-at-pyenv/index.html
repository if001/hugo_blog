    • pyenv環境でaws cli入れたらコマンドが見つからないって言われたので調べたメモ

    バージョン：pyenv:1.0.10

    公式通りにインストールする。

    $ pip3 install awscli --upgrade --user

    すると、

    $ aws
-bash: aws: command not found

    コマンドが見つからない….

    調べると、$HOME/.localにインストールされるらしいので、パスを通す。

    aws --version
aws-cli/1.16.29 Python/2.7.13 Darwin/16.7.0 botocore/1.12.19

    できた！

    pyenvが問題なのかはわからん….
